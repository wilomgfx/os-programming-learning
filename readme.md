## Build the image
`docker build -t log710c:v1 .`
## Run the container
### MacOs/Linux
open a shell and run

` docker run -it -v $PWD:/workspace log710c:v1`

### Windows
open powershell and run

` docker run -it -v $PWD:/workspace log710c:v1`
